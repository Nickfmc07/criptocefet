Cifra de Vigenère

A Cifra de Vigenère foi inventada por Giovan Battista Bellaso e nomeada em homenagem a Blaise de Vigenère. Para saber mais sobre a historia dessa cifra, clique aqui. O método de criptografia consiste em definir uma chave, pode ter qualquer tamanho, e usando a tabula recta deslocar cada letra da mensagem original por um número n definido pela chave.

Para que o método funcione, é necessário que o comportamento dele seja como o de uma função sobrejetora, ou seja, cada letra na mensagem original deve corresponder a pelo menos uma letra na mensagem cifrada. Pra garantir isso, o comportamento do deslocamento de cada letra ser cíclico, como um relógio, onde após a letra "Z" voltamos para a letra "A". Por exemplo, se deslocarmos a letra "Z" 1 casa para a direita, obteremos "A":


Para garantir que a cifra se comporte como um relógio, sempre que operarmos com números maiores que 25, devemos aplicar o módulo 26. Assim, 26 mod 26 é 0, 27 mod 26 é 1, e assim por diante. Por exemplo:



\( 27 \bmod 26 = 1 \)

Ou seja, 27 casas para a direita é o mesmo que 1 casa para a direita. Então 27 representa a letra "B", assim como 1. Portanto, podemos dizer que o deslocamento é cíclico e se comporta como uma função sobrejetora.

Este guia mostra como cifrar e decodificar usando a Cifra de Vigenère. Veja a tabela abaixo:



Escolha a chave


Para usar a cifra, vamos primeiro escolher qual chave vamos usar. A chave deve ser conter apenas letras. Ele pode ter qualquer tamanho.


Cifrar e Decifrar


Agora, o que vamos fazer é deslocar cada letra da mensagem original pelo número de casas definido pela chave. Se a primeira letra da chave for "B", que é a segunda letra do alfabeto, vamos deslocar a primeira letra da mensagem original 1 casa para a direita. Dizendo que a letra era "F", ao deslocá-la 1 casa para a direita, obtemos "G".

Usando a tabula recta, isso fica mais fácil. Usa a letra original como coluna e a letra da chave como linha. O ponto de encontro é a letra cifrada.

Para cada letra da mensagem original, usamos a próxima letra da chave. Se a chave for menor que a mensagem, voltamos ao início da chave.
