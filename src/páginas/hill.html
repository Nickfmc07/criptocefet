Cifra de Hill

A cifra de Hill foi criada por Lester S. Hill em 1929, com o objetivo de tornar a análise de padrões mais difícil de ser praticada. Ela utiliza matrizes para criptografar mensagens, tornando-as mais seguras.

1º passo: Escolher a matriz chave 2x2: A matriz chave 2x2 é selecionada, sendo importante que seja invertível (determinante diferente de zero).

2º passo: Converter a mensagem em números: A mensagem é convertida para números (A = 0, B = 1, ..., Z = 25). Se necessário, adiciona-se uma letra de preenchimento.

3º passo: Multiplicar a matriz chave pela mensagem: Cada par de números (bloco de 2 letras) é multiplicado pela matriz chave, resultando na mensagem cifrada (em números).

4º passo: Converter a mensagem cifrada em letras: Os números obtidos são convertidos de volta para letras, formando a mensagem criptografada.

5º passo: Descriptografar a mensagem: O destinatário calcula a matriz chave inversa e a multiplica pela mensagem cifrada, restaurando os números originais, que são então convertidos de volta em letras para recuperar a mensagem original.
